About this portfolio

How it works

This page is a single static HTML file that runs entirely in the browser. It uses the public GitHub API to fetch files from Jzelfthr/DEEP_LEARNING. Markdown and notebook markdown cells are rendered into the preview panel; HTML files are shown as full pages inside the panel using a sandboxed iframe.

That means you can add or update notebooks, markdowns, and HTML pages in the repo and the portfolio will automatically reflect those changes the next time it loads.
