This page gives a structured, interactive overview of my work in deep learning. It reads a public GitHub repo (DEEP_LEARNING) and surfaces lectures, labs, exercises, and reflections in a Jupyter-Book-style layout.

What You Will Find

About this portfolio

This HTML page is completely static and works purely in the browser. When hosted on GitHub Pages, it calls the public GitHub API to fetch directory listings and markdown files from Jzelfthr/DEEP_LEARNING. No server-side code or build step is required.

For large notebooks, the page links out to GitHub’s viewer instead of rendering everything here. If the repository remains public, the viewer will keep working with future commits.
