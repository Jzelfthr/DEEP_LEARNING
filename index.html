About this portfolio

How it works

This page is a single static HTML file that runs entirely in the browser. When hosted on GitHub Pages, it calls the public GitHub API to fetch directory listings and files from Jzelfthr/DEEP_LEARNING. Markdown files and notebook markdown cells are rendered directly into the preview panel.

This makes the portfolio easy to maintain: whenever the repository is updated with new lectures, labs, or reflections, the site automatically reflects those changes the next time it is loaded.
